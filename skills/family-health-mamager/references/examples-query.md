# Examples: Query

## 示例 1：药物查询

用户问："妈妈现在在吃什么药？"

正确流程：
1. 读取该成员的 `medications` tab
2. 筛选 `状态` 为 `使用中` 或 `按需` 的记录
3. 先给结论，再附证据

输出示例：

```md
## 结论
妈妈当前在使用 2 种药物：左甲状腺素钠片（优甲乐）和维生素D3。

## 证据
- 左甲状腺素钠片 50μg/日，自 2025-08-01 起使用中 [1]
- 维生素D3 800IU/日，自 2026-01-15 起使用中 [2]

## 引用来源
[1] 2025-08-01，来源文件：records/2025/2025-08-01-内分泌复诊/门诊病历.pdf
[2] 2026-01-15，来源文件：records/2026/2026-01-15-营养科门诊/门诊病历.pdf
```

## 示例 2：检查趋势查询

用户问："爸爸最近几次 TSH 结果怎么样？"

正确流程：
1. 读取该成员的 `measurements` tab
2. 筛选 `检查细项` 包含 `促甲状腺激素` 或 `TSH` 的记录
3. 按 `检查日期` 排序
4. 先给趋势结论，再按时间列数值

输出示例：

```md
## 结论
爸爸近 3 次 TSH 结果呈下降趋势，最近一次已回到参考范围内。

## 证据
- 2025-06-10：TSH 8.2 mIU/L（参考 0.27–4.20），偏高 [1]
- 2025-12-05：TSH 5.1 mIU/L（参考 0.27–4.20），偏高 [2]
- 2026-03-12：TSH 3.8 mIU/L（参考 0.27–4.20），正常 [3]

## 引用来源
[1] 2025-06-10，来源文件：records/2025/2025-06-10-内分泌初诊/甲功五项.pdf
[2] 2025-12-05，来源文件：records/2025/2025-12-05-内分泌复诊/甲功五项.pdf
[3] 2026-03-12，来源文件：records/2026/2026-03-12-内分泌复诊/甲功五项.pdf
```

## 示例 3：报告定位

用户问："去年体检的超声报告在哪？"

正确流程：
1. 读取 `timeline`，筛选上一年度 `事件类型` 为 `体检` 的记录
2. 找到对应 `归档路径`
3. 在该事件目录下查找超声相关文件

输出示例：

```md
## 结论
去年体检的超声报告归档在以下位置：

- 甲状腺超声：`records/2025/2025-05-10-年度体检/甲状腺超声.pdf`
- 腹部超声：`records/2025/2025-05-10-年度体检/腹部超声.pdf`

## 引用来源
[1] 2025-05-10，归档路径：records/2025/2025-05-10-年度体检/
```
